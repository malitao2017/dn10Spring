spring的主要内容

1.spring IOC(DI依赖注入来实现)类的获取，主要两种：ApplicationContext 和 BeanFactory

2.spring 基础，bean的配置： scope(单例/多例),init-method(类的初始化方法),destory-method(销毁方法),lazy-init(懒加载)

3.spring AOP： 1.配置切面进行日志管理 2.事务处理

4.spring对数据库的支持：1.支持spring jdbc 2.支持hibernate

5.动态代理： 1.有接口的话，使用JDK自带的动态代理即可。 2.没有接口，则使用Spring AOP的CGLIB
